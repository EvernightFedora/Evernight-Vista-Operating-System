## <font color=#2f81f7>概要</font>

Evernight Vistaは、Fedora Linuxをベースにしたオープンソースのオペレーティングシステムです。『崩壊：スターレール』をテーマにしたLinuxディストリビューションです。

Fedora Linuxは優れたLinuxディストリビューションであり、迅速で安定して安全です。古いコンピューターでも非常に優れた体験ができます。しかし、純粋なコマンドラインによるインストールプロセスはやや複雑で、十分なコンピューター知識が必要です。そのために時間、労力、忍耐を費やさなければならず、順調にインストールが完了しても、グラフィカルユーザーインターフェースのインストール、ソフトウェアリポジトリの設定、入力メソッドのインストール、グラフィックカードドライバのインストールなど一連の設定作業に直面します。これらすべてを完遂するのに十分な時間、労力、忍耐を持つ人はそう多くありません。多くの初心者はこの困難なプロセスによってFedora Linuxの新しい世界への門前で阻まれるかもしれません。これがまさにEvernight Vista誕生の理由です。Fedora Linuxの利点を引き継ぎつつ、インストールを簡単にし、すぐに使えるようにし、よりユーザーフレンドリーにしました。複雑なインストールや設定を必要とせず、Fedora Linuxの魅力をすぐに体験できます！


## <font color=#2f81f7>Evernight Vista を選ぶ理由</font>

### 🔐 <font color=#2f81f7>強力で安全</font>
Evernight Vista はデフォルトで SELinux を使用。もちろん、Evernight Vista はセキュアブートにも対応しています！

### 💻 <font color=#2f81f7>簡単にインストール</font>
Evernight VistaはAnacondaインストーラーを使用しています。インターフェースは使いやすく、シンプルです。

### 🏪 <font color=#2f81f7>ワンストップソフトウェアエコシステム</font>
Evernight Vista は RPM パッケージ、flatpak、linglong をサポートしています。よく使うソフトウェアをワンストップで入手できます。

### 🆙 <font color=#2f81f7>使いやすさ</font>
Evernight Vistaは優れたオペレーティングシステムのユーザー体験を提供することを目指しており、ユーザーの技術的な背景に関係なく、誰でも簡単にEvernight Vistaを使いこなせると私たちは信じています。

### 🚀 <font color=#2f81f7>速度と信頼性</font>
Fedora Linuxの優れた特性のおかげで、Evernight Vistaは軽量で高速、安定したシステムです。Evernight Vistaは新しいSCXスケジューリングエンジンを使用しており、さらに高性能です！
